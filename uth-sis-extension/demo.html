🎓 UTH SIS Grades Extension

Αυτόματη εξαγωγή βαθμών από το σύστημα SIS του Πανεπιστημίου Θεσσαλίας

📋 Τι κάνει το Extension

  • ✅ Εξάγει αυτόματα βαθμούς από τη σελίδα του SIS
  • ✅ Αποθηκεύει τους βαθμούς τοπικά στον browser
  • ✅ Εξάγει σε JSON format για χρήση σε άλλες εφαρμογές
  • ✅ Ελληνικό interface
  • ✅ Δεν αποθηκεύει κωδικούς πρόσβασης

🔧 Εγκατάσταση

Βήμα 1: Ανοίξτε τον Chrome/Edge
Βήμα 2: Πηγαίνετε στο chrome://extensions/
Βήμα 3: Ενεργοποιήστε το "Developer mode" (επάνω δεξιά)
Βήμα 4: Κάντε κλικ στο "Load unpacked"
Βήμα 5: Επιλέξτε το φάκελο uth-sis-extension

📖 Χρήση

1. Ανοίξτε τη σελίδα του SIS: https://sis.uth.gr
2. Συνδεθείτε με τα credentials σας
3. Πλοηγηθείτε στη σελίδα των βαθμών
4. Κάντε κλικ στο εικονίδιο του extension
5. Κάντε κλικ στο "Εξαγωγή από τρέχουσα σελίδα"

📊 Παράδειγμα Αποτελέσματος

Το extension θα εξάγει βαθμούς σε αυτή τη μορφή:

[ { "course": "Εισαγωγή στην Πληροφορική", "grade": 8.5, "semester": "1ο Εξάμηνο", "extractedAt": "2024-01-15T10:30:00.000Z" }, { "course": "Προγραμματισμός Ι", "grade": 9.0, "semester": "1ο Εξάμηνο", "extractedAt": "2024-01-15T10:30:00.000Z" } ]

Προβολή βαθμών:

Εισαγωγή στην Πληροφορική

8.5

Εξάμηνο: 1ο Εξάμηνο

Προγραμματισμός Ι

9.0

Εξάμηνο: 1ο Εξάμηνο

Μαθηματικά Ι

7.5

Εξάμηνο: 1ο Εξάμηνο

Δομές Δεδομένων

8.0

Εξάμηνο: 2ο Εξάμηνο

🔗 Ενσωμάτωση με Frontend

Μετά την εξαγωγή, μπορείτε να ανεβάσετε το JSON αρχείο στο frontend:

1. Εξάγετε τους βαθμούς με το extension
2. Κατεβάστε το JSON αρχείο
3. Πηγαίνετε στο frontend της εφαρμογής
4. Ανεβάστε το JSON αρχείο για προβολή

⚠️ Σημαντικές Σημειώσεις

  • Το extension λειτουργεί μόνο σε σελίδες του SIS (sis.uth.gr)
  • Δεν αποθηκεύει κωδικούς πρόσβασης
  • Όλα τα δεδομένα αποθηκεύονται τοπικά
  • Απαιτείται σύνδεση στο SIS για να λειτουργήσει
  • Αν αλλάξει η δομή της σελίδας, ίσως χρειαστεί ενημέρωση
